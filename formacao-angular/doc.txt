comandos:
ng serve = roda a aplicação
ng test = testa a aplicação
ng e2e = executa testes unitarios, simula operações de usuario utilizando sistema
ng lint = valida a qualidade do codigo
ng build --prod = gera diretorio dist, que contem um codigo pronto e otimizado para ir pra produção
ng new nome_projeto = cria um projeto com angular CLI
npm install = instala nodel_modules
